##题目1 相邻数对
- 时间限制: 1 秒
- 空间限制: 256 MB
- 问题描述
</br>给定n个不同的整数，问这些数中有多少对整数，它们的值正好相差1。
- 输入格式
</br>输入的第一行包含一个整数n，表示给定整数的个数。
</br>第二行包含所给定的n个整数。
- 输出格式
</br>输出一个整数，表示值正好相差1的数对的个数。
- 输入样例
</br>6
</br>10 2 6 3 7 8
- 输出样例
</br>3
- 样例说明
</br>值正好相差 1 的数对包括(2, 3), (6, 7), (7, 8)。
</br>评测用例规模与约定1<=n<=1000，给定的整数为不超过 10000 的非负整数。
